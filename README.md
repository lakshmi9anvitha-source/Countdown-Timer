# Countdown / Pomodoro Timer (Python)

A simple command-line countdown timer with an optional Pomodoro mode.  
Great for productivity, studying, or time-based tasks.

---

##  Modes Available
### 1. Countdown Mode
Enter any number of minutes and the timer counts down with a live MM:SS display.

###  Pomodoro Mode
Runs cycles of:
- Work session  
- Break session  

Fully customizable (work minutes, break minutes, number of cycles).
